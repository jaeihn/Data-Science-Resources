Description

View staging area

Add updates and deletion of files

git add .

git add -A

Add everything available to staging area

git add -p

Add with interactive prompt for each file (y/n)

git diff --staged

Compare the changes in the staging area

git diff {older_hash}

git diff {file1} {file2}

git log

git log --oneline

View commit history (minimal)

git log --graph --all --oneline

Draw tree graph of commit history, all branches

git switch {branch}

Switch between branches

git switch -c {branch}

Create and switch to branch

git merge {branch}

Merge branch into main

git branch -d {branch}

Delete branch

git push --set-upstream origin {branch}

Push a new branch onto remote repo

git revert

Create a new commit that reverts previous commits

git reset --soft {hash}

Uncommit, changes are left staged

git reset --mixed {hash}

git reset --hard {hash}

git reset {hash}

git push -f origin

Force reset commit to remote repo (deletes any commit history in between)
